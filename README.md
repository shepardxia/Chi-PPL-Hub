# Chi PPL Hub

A collection of PPL resources for Chi projects — tutorials, code examples, and evaluation data for domain-specific probabilistic programming languages.

## Languages

| Language | Description |
|----------|-------------|
| [WebPPL](webppl/) | Probabilistic programming embedded in JS |
| [Memo](memo/) | Probabilistic programming embedded in Python |
| [Pluck](pluck/) | Probabilistic programming in a Lisp-like language |
